About This Project

Medical AI Assistant

A final year project demonstrating the application of natural language processing in healthcare education.

Project Overview

This Medical AI Assistant was developed as a final year project to explore how artificial intelligence can be used to provide educational information about general health topics. The system uses advanced natural language processing techniques to understand health-related questions and generate informative responses.

Key Features
  • Natural language understanding for medical queries
  • Educational responses based on reliable health information
  • Safety filters to ensure appropriate information sharing
  • User-friendly interface for easy interaction
Technology Stack

This project was built using the following technologies:

  • Backend: Python, Flask
  • Natural Language Processing: Transformers, NLTK, SpaCy
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Deployment: Heroku
Limitations and Ethical Considerations

This assistant is designed for educational purposes only and is not a substitute for professional medical advice, diagnosis, or treatment. The system has several important limitations:

  • It cannot diagnose medical conditions
  • It does not have access to personal medical information
  • It may not be aware of the latest medical research
  • It cannot prescribe medications or treatments

Always consult with qualified healthcare professionals for medical concerns.
